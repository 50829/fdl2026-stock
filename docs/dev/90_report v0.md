  1. 问题定义
      - 输入：历史日频量价、技术指标、截面特征。
      - 输出：股票未来收益排序分数。
      - 目标：不是预测绝对涨跌幅，而是预测横截面排

      - 核心标签：
          - label1d：更适合每日滚动持仓、短周期换
            仓。

          - label5d：更适合低频滚动持仓、收益更平
            滑。

  2. 数据与股票池
      - 说明 features.parquet 不是全市场原始数据，而
        是已经经过股票池和流动性过滤后的训练特征。

      - universe.parquet 是股票池资格表，用来决定哪
        些股票进入训练和预测。

      - 报告里必须明确：目前“无流动性过滤预测”只是实
        时诊断，不等于训练了一个真正无流动性过滤模
        型。

  3. 模型实验
      - 写清楚模型路线：
          - 深度模型：MLP / GRU / TCN 等作为探索。
          - 主力有效模型：LightGBM / XGBoost。
          - 融合模型：label1d residual-rank MLP 最终
            alpha 选择为 0.0，所以实盘 label1d 主模
            型实际是 LightGBM。

      - 重点指标：IC、ICIR、MSE、策略收益、夏普、最
        大回撤。

  4. 策略实验
      - label1d 应该对应 topk step_days=1 的每日滚动
        持仓。

      - label5d 更适合 rolling tranche 或较低频持
        仓。

      - 报告要解释：
          - topk20_drop3 / topk20_drop5
          - rank buffer
          - rolling tranche
          - 防守策略：市场压力降仓 + 组合回撤控制。

  5. 核心结果
      - label5d 的信号稳定性更好，ICIR 和保守策略表
        现更稳。

      - label1d 在高换手 topk 策略里收益更强，但回撤
        和换手压力更大。

      - 2024 年是所有策略的压力测试年份，普通策略出
        现约 40% 回撤，防守策略能显著降低回撤，但会
        牺牲收益。

      - walk-forward 结果说明：all_windows 更稳，
        short_5_10 更激进，收益高但回撤也更大。
