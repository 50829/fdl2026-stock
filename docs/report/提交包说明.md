## 未包含内容

| 类别 | 原因 |
| --- | --- |
| `data/raw/` | 原始数据体积较大，且可能包含外部来源数据，不适合直接放入课程报告包 |
| `data/processed/*.parquet` | 约 2.7G，属于本地复现实验数据，不是报告审阅必需内容 |
| 完整 `outputs/models/` | 约 9.7G，包含大量预测 parquet、checkpoint 和中间实验产物 |
| 完整 `outputs/strategy/` | 约 917M，包含历史回测曲线、交易明细和中间版本 |
| 缓存目录 | `.pytest_cache/`、`__pycache__/` 等不影响复现 |

如需完整复现训练，需要额外准备 `data/processed/`，并按 `README.md` 中的命令重新生成模型和策略产物。

## 报告仍需人工补充

以下内容无法由代码自动生成，需要提交前人工补齐：

1. 三位组员姓名、学号和分工。
2. 同花顺模拟交易收益趋势截图。
3. 每日调仓记录截图。
4. 每日持仓记录截图。
5. 模拟交易最终收益率。
6. 实际交易与模型推荐是否一致的说明。
